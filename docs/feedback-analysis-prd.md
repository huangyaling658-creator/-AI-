# 反馈分析模块 — 产品需求文档 (PRD)

## 1. 概述

### 1.1 产品背景
产品经理日常需要处理来自多渠道的用户反馈（App Store评论、客服工单、社群讨论、用研访谈等），手动整理耗时且容易遗漏。本模块通过 AI 自动完成反馈的解析、分类、聚合和可视化，帮助 PM 快速定位需求痛点，支撑决策和向上汇报。

### 1.2 目标用户
产品经理、用户研究员、运营人员

### 1.3 核心价值
- 多格式文件"扔进来就能分析"，零门槛
- AI 自动推荐分析维度，减少人为遗漏
- 图表化结论，可直接用于汇报

---

## 2. 功能流程

```
上传文件（支持多份）
  → 数据解析 + 合并预览
  → AI 读取数据，推荐分析维度
  → 用户选择/新增维度，点击开始分析
  → 分析结果展示 + 标注数据下载
  → 图表化呈现分析结论
  → 导出完整报告
```

---

## 3. 详细需求

### 3.1 Step 1：上传文件

**功能描述：** 用户上传一份或多份待分析的反馈数据文件。

**支持格式（按推荐程度排序）：**

| 格式 | 说明 |
|------|------|
| CSV / Excel (.xlsx) | 最佳，结构化数据，字段清晰 |
| TXT / Markdown | 纯文本，直接处理 |
| PDF（文本型） | 提取文本后处理 |
| Word (.docx) | 提取文本后处理 |
| 图片 (PNG/JPG) | 利用模型视觉能力识别文字 |

**交互细节：**
- 拖拽上传区域，支持点击选择或拖拽文件
- 支持同时上传多份文件（最多10份）
- 上传后显示文件列表：文件名、格式、大小、状态（解析中/成功/失败）
- 单份文件大小限制：20MB
- 上传完成后自动触发解析

### 3.2 Step 2：数据预览与确认

**功能描述：** 系统解析上传文件，展示数据预览供用户确认。

**单文件场景：**
- 展示前 10 条数据预览
- 对于表格类文件，以表格形式展示
- 对于文本类文件，展示前 N 段内容

**多文件场景：**
- AI 自动识别每份文件中的"反馈内容"字段（如"评论"、"反馈"、"内容"等列）
- 展示字段映射关系：每份文件 → 被识别的反馈字段
- 用户可手动修正字段映射
- 合并后展示统一预览（前 10 条）
- 显示合并统计：总条数、来源分布

**操作：** 用户确认数据无误后点击"下一步"

### 3.3 Step 3：选择分析维度

**功能描述：** AI 基于数据内容，2秒内推荐可能的分析维度。

**AI 推荐维度（示例）：**
- 情感倾向（正面/负面/中立）
- 问题分类（功能缺陷、体验问题、性能问题、需求建议等）
- 优先级评估（基于频次 × 情感强度）
- 功能模块归属（反馈涉及产品哪个模块）
- 用户画像标签（如果数据包含用户信息）
- 时间趋势（如果数据包含时间戳）

**交互细节：**
- 每个维度以卡片形式展示，包含：维度名称、简要说明、数据是否支持该维度的提示
- 卡片可点击选中/取消，默认推荐的维度预选中
- 提供"新增自定义维度"输入框，用户可自行添加分析角度
- 不支持的维度（如数据中无时间戳但推荐了时间趋势）标记为灰色并提示原因

### 3.4 Step 4：分析执行

**功能描述：** 用户选择维度后点击"开始分析"，10秒内返回结果。

**分析方法：**
- **语义聚类：** AI 理解每条反馈的语义，按相似性归组
- **情感分析：** 每条反馈打情感分（正面/负面/中立 + 强度 1-5）
- **频次统计：** 相似问题出现次数统计
- **关键词提取：** 每个聚类提取代表性关键词
- **摘要生成：** 每个分类生成一句话摘要

**数据清洗（在分析前自动执行）：**
- 去除重复反馈
- 过滤无意义内容（如纯表情、"好的"、"嗯"等）
- 统一编码和格式
- 合并极度相似的反馈

**交互细节：**
- 分析过程中显示进度条和当前步骤提示
- 步骤提示示例：数据清洗中 → 语义分析中 → 聚类归组中 → 生成结论中

### 3.5 Step 5：分析结果 + 标注数据下载

**功能描述：** 展示分析结果摘要，并提供标注后数据的下载。

**结果摘要：**
- 总反馈数 / 有效反馈数 / 去重后数量
- 各分类下的反馈数和占比
- 每条反馈被标注：所属分类、情感倾向、关键词

**下载功能：**
- 标注后的数据可下载
- 下载格式与上传格式一致（上传CSV则下载CSV，上传Excel则下载Excel）
- 新增标注列：分类、情感、关键词、优先级分数

### 3.6 Step 6：图表化呈现

**功能描述：** 用图表呈现分析结论，便于向上汇报。

**图表类型：**

| 图表 | 展示内容 |
|------|----------|
| 饼图 | 反馈分类占比 |
| 柱状图 | 各分类反馈数量对比 |
| 情感分布图 | 正面/负面/中立比例 |
| 词云图 | 高频关键词可视化 |
| Top N 列表 | 高频问题排行（含频次、情感、趋势） |

**原文溯源：**
- 图表中的每个数据点/分类可点击展开
- 展开后显示该分类下的原始反馈列表
- 支持在原文中高亮匹配的关键词

### 3.7 Step 7：导出报告

**功能描述：** 一键导出包含图表和结论的完整分析报告。

**导出格式：** PDF

**报告内容：**
- 分析概要（数据来源、样本量、分析时间）
- 各维度分析结论（含图表）
- 高频问题 Top 10
- 行动建议摘要

---

## 4. 非功能需求

| 项目 | 要求 |
|------|------|
| 分析响应时间 | 维度推荐 ≤ 2秒，完整分析 ≤ 10秒（500条以内） |
| 文件大小限制 | 单文件 ≤ 20MB |
| 文件数量限制 | 单次最多 10 份 |
| 数据安全 | 用户 API Key 本地存储，文件不持久化到服务器 |
| 浏览器兼容 | Chrome / Edge / Safari 最新版 |

---

## 5. MVP 范围

**MVP 包含：**
- 上传（CSV / Excel / TXT / PDF / Word / 图片）
- 多文件合并预览
- AI 推荐分析维度 + 自定义维度
- 语义聚类 + 情感分析 + 频次统计
- 标注数据下载
- 图表呈现（饼图、柱状图、情感分布、词云、Top N）
- 原文溯源
- PDF 报告导出

**后续迭代：**
- 多批次对比分析（本月 vs 上月）
- 分析历史记录
- 团队协作（分享分析结果）
